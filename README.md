# MealPreppy 

MealPreppy is a full-stack meal planning platform designed to simplify one of the most common yet overlooked problems: **deciding what to cook and managing groceries efficiently for the week**.

## Problem Statement

Planning meals daily is time-consuming and inconsistent. Users often:
- Struggle to decide what to cook
- End up repeating meals or making unhealthy choices
- Buy groceries inefficiently, leading to waste or missing ingredients
- Lack a structured weekly planning system

##  Solution

MealPreppy transforms this process into a **structured, data-driven workflow**:

1. Users explore a variety of meals (veg/non-veg)
2. Select meals for the week or choose curated plans
3. The system automatically generates:
   - A **weekly meal plan**
   - A **consolidated grocery list**
4. Users can directly proceed to order, eliminating planning friction

This bridges the gap between **meal decision-making and execution**.

##  Key Highlights

- **End-to-end workflow**: Meal selection - Plan generation - Grocery aggregation - Order
- **Data-driven aggregation**: Ingredients from multiple meals are intelligently combined
- **Flexible planning**: Supports both manual selection and curated plans
- **Real-time interaction**: Immediate feedback on selections and plan generation
- **User-centric design**: Focused on reducing cognitive load and decision fatigue

## Tech Stack

- **Frontend**: React (Vite)
- **Backend**: Node.js + Express
- **Database**: MongoDB Atlas
- **Deployment**: Vercel + Render
